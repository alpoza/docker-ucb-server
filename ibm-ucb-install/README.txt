Welcome to the IBM UrbanCode Build Server Installer

Requirements:
1. To install IBM UrbanCode Build you will need to have Java 5 or later
   installed.
2. To use a external database you will need to have access to a Oracle, MySQL
   or SQL Server database with a user account capable of creating tables. If
   you do not have this, uBuild can be installed with a embedded Derby
   database. It is not recommended to use the embedded database for production.
   
Installation:
1. On Windows, execute 'install-server.bat' from the command line.
    - or -
   On Linux or Unix, execute './install-server.sh' from the command line.
